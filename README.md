🛒 Shop-Mag

Simple online shop project built with modern frontend technologies.
This project demonstrates basic e-commerce functionality and component-based architecture.

🚀 Features
Product listing
Shopping cart functionality
Responsive UI
Component-based structure
Clean and scalable code
🛠️ Tech Stack
React
JavaScript (ES6+)
HTML5
CSS3

⚙️ Installation

Clone the repository:

git clone https://github.com/m1s0n1/shop-mag.git

Go to the project folder:

cd shop-mag

Install dependencies:

npm install

Run the project:

npm start
